# 🎬 عرض تسويقي — منظومة الماكينة الانتخابية

عرض HTML سينمائي متحرّك يروّج للمنظومة. مبنى على هوية **Command Deck** الرسمية للمشروع
(حبري داكن + كهرماني حاسم + تيل إيجابي + أرقام Mono جدولية).

## ▶️ التشغيل

```bash
# الخيار 1: افتح الملف مباشرة في المتصفح
start index.html          # Windows
open index.html           # macOS

# الخيار 2: عبر خادم بسيط (للتسجيل الأنظف)
npx serve promo
```

## 🎥 تحويله إلى فيديو (.mp4)

العرض مُصمّم خصيصاً ليُسجَّل كفيديو تسويقي. الطريقة:

### عبر OBS Studio (الأفضل، مجاني)
1. ثبّت [OBS Studio](https://obsproject.com/).
2. أضف مصدر **Display Capture** أو **Window Capture** (نافذة المتصفح).
3. اضبط الدقة على **1920×1080** ومعدّل الإطارات **60fps**.
4. افتح `index.html` في المتصفح بملء الشاشة (`F11`).
5. اضغط زر التشغيل التلقائي **▶** في العرض (يتقدّم مشهد كل 5 ثوانٍ).
6. ابدأ التسجيل في OBS → دعه يكمل الـ 6 مشاهد (~30 ثانية) → أوقف.
7. صدّر بصيغة MP4 (Settings → Output → Recording Format → mp4).

### عبر Windows Game Bar (الأسرع)
1. افتح `index.html` في متصفح بملء الشاشة.
2. اضغط **Win + G** لفتح Game Bar.
3. اضغط زر التسجيل (أو Win + Alt + R).
4. شغّل العرض التلقائي **▶** حتى النهاية.
5. أوقف التسجيل — يُحفظ في `Videos/Captures/`.

### عبر FFmpeg (للأتمتة، متقدّم)
```bash
ffmpeg -f gdigrab -framerate 30 -i desktop -t 35 -c:v libx264 -preset slow -crf 18 promo.mp4
```

## 🎛️ التحكم في العرض

- **‹ / ›** : التنقّل بين المشاهد (RTL: يسار = تالي).
- **▶ / ⏸** : تشغيل تلقائي (مشهد كل 5 ثوانٍ).
- **Space** : تشغيل/إيقاف تلقائي.
- **F11** : ملء الشاشة (موصى به قبل التسجيل).

## 📐 الهوية البصرية المستخدمة

مطابقة لـ `DESIGN_SYSTEM.md`:
- **الألوان:** خلفية `#0B1120`، إشارة كهرمانية `#F2A024`، ثانوي تيل `#2DD4BF`.
- **الخطوط:** IBM Plex Sans Arabic (الواجهة) + IBM Plex Mono (الأرقام الجدولية).
- **العنصر المميِّز:** شارة "مباشر" النابضة + أرقام المؤشرات الكبيرة بأشرطة حالة دقيقة.
- **الاتجاه:** RTL كامل، كثافة عالية، hairlines.

## ✏️ التخصيص

- **الأرقام:** عدّل `data-to` وعرض `width` في `kpi-bar` بمشهد 1.
- **النصوص:** عدل مباشرة في وسوم `<h1>`/`<p>`/`<h2>`.
- **الألوان:** عدل متغيّرات `:root` أعلى الملف.
- **مدة المشهد التلقائي:** عدل `5000` في `setInterval(next, 5000)`.
- **عدد المشاهد:** أضف/احذف وسوم `<section class="scene" data-scene="N">` وحدّث `total` لو لزم.
